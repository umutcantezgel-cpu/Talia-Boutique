# Talia Boutique - Premium Next.js Application

Willkommen im Repository von **Talia Boutique**, einem maßgeschneiderten, luxuriösen E-Commerce-Erlebnis, optimiert für Performance, Zugänglichkeit und SEO.

## Features
- **Next.js 15+ (App Router):** Modernste React-Architektur für schnellste Ladezeiten.
- **Premium Design:** Individuelles, auf Hex-Codes basierendes Farbschema (`#3B1C32`, `#D9A5B3`, `#FDFBF7`, `#222222`).
- **SEO & Crawlability:** Vollständig optimiert mit `robots.ts`, `sitemap.ts` und strukturierter Schema.org-Datengenerierung für Local SEO (Wetzlar).
- **Gamifizierte UI:** Interaktive Cookie-Banner (Slide-to-Accept), Custom Cursor Glows und weiche Page-Transitions.

## Deploy on Vercel

Dieses Projekt ist für ein **One-Click Deploy** auf Vercel konfiguriert. 

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fumutcantezgel-cpu%2FTalia-Boutique)

1. Verbinde dein GitHub-Repository mit Vercel.
2. Vercel erkennt automatisch das Next.js Framework.
3. Füge ggf. notwendige Environment-Variablen (`.env.local`) im Vercel Dashboard hinzu.
4. Klicke auf **Deploy**.

## Run Locally

**Voraussetzungen:** Node.js (Version 18+)

1. Installiere die Abhängigkeiten:
   ```bash
   npm install
   ```
2. Starte den Entwicklungsserver:
   ```bash
   npm run dev
   ```
3. Öffne [http://localhost:3000](http://localhost:3000) im Browser.
